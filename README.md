# GWS
GWS (Go Web Server) - A simple web server for serving files in GO.

Currently in version 0.3, 

## use GWS:

### Ports
```
./gws -p="<port number>"
```
e.g. 
```
./gws -p="5001"
```
### Set Directory
```
./gws -d="<directory>"
```
e.g. 
```
./gws -p="../"
```

### Toggle SSL on
```
./gws -s
```

**N.b This will default to port 8000 if not specified**
